# Data-Structures
Contains implementation of major data structures
dspd_ll.c dspd_ll.h dspd.c contains the linked list implementation of a phonebook implemented using singly linked list.
Features of the phonbeok : 
1. Insert Contact(Personal/Professional). 
2. Delete Contact(Personal/Professional).
3. Edit Contact(Personal/Professional).
4. View all personal contact in sorted order.
5. View all professional contact in sorted order.
6. Search for a contact(Personal/Professional).
7. Remove duplicate contacts(Personal/Professional).
8. Merge two contact lists(Personal/Professional).
9. Print the phonebook details owned by a particulaer person in sorted order. (Prints both personal and professional data).
